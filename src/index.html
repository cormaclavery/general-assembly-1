This site

As I am already familiar with HTML, CSS and JS from previous online tutorials on Treehouse, Coursera, Codeacademy I decided I would make use of some additional development tools/frameworks that I researched during the course of this project. This site was built using the Material Design lite framework which adds the material design style to the web. MDL was applied to the buttons and to the cards but I did not make too much use of it as I wanted to match the style of the example site built on the dash course provided. I also installed some development tools using NPM and implemented in a Gulp file.

Through Gulp I applied the following utilities:

  • Browser-sync for testing on the fly design changes
  • SASS to allow nesting of css style components and implementation of variables in css i.e. scss
  • Minify to make the converted css in the build neater/smaller file size
  • Auto-prefixer to automatically apply other browser kits to allow cross/back compatibility

All changes have been recorded in GitHub and a link to the Git has been provided in the navbar above and will open in a new tab when selected. I used SourceTree to manage all commits.
